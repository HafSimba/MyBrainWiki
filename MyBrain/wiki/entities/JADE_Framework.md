# JADE Framework

## Profilo
Java Agent Development Framework (JADE), framework storico per sviluppo di sistemi multi-agente con supporto a standard di interoperabilita.

## Ruolo nel dominio
Serve come riferimento implementativo per prototipare e standardizzare applicazioni MAS in contesti ingegneristici.

## Fatti rilevanti
- La fonte lo cita come framework nato per accelerare sviluppo e standardizzazione di sistemi MAS ([SRC-20260417-multi-agent-system-wikipedia](../sources/SRC-20260417-multi-agent-system-wikipedia.md)).
- Viene menzionato insieme a standard di riferimento nell'ecosistema agenti (FIPA/OMG) ([fonte raw](../../raw/sources/2026-04-17--multi-agent-system-wikipedia.md)).
