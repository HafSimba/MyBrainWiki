# Bash

## Profilo
GNU Bourne-Again SHell, shell testuale diffusa in ambienti Linux per interazione da terminale e scripting.

## Ruolo nel dominio
Fornisce il layer operativo per eseguire comandi built-in, orchestrare utility di sistema e costruire automazioni tramite script e pipeline.

## Fatti rilevanti
- La fonte presenta bash come base della command line Linux, affiancata da un ampio set di utility di sistema ([SRC-20260417-a-z-index-linux-command-line](../sources/SRC-20260417-a-z-index-linux-command-line.md)).
- I comandi built-in bash sono esplicitamente distinti dai comandi esterni ([fonte raw](../../raw/sources/2026-04-17--a-z-index-linux-command-line.md)).
- Nell'indice compaiono strumenti chiave per shell automation, gestione processi, rete e file system ([fonte raw](../../raw/sources/2026-04-17--a-z-index-linux-command-line.md)).
