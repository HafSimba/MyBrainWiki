# Indice Comandi CMD Windows

## Definizione
Mappa operativa A-Z dei comandi CMD per trovare rapidamente utility e sintassi di base su sistemi Windows.

## Perche conta
Riduce il tempo di ricerca durante troubleshooting e automazione, soprattutto quando serve orientarsi tra categorie diverse di comandi amministrativi.

## Regole operative
- Verifica sempre se un comando e interno CMD o esterno prima di usarlo fuori dal prompt classico.
- Testa i comandi potenzialmente distruttivi in ambienti controllati o su dati non critici.
- Per task ricorrenti, trasforma comandi manuali in script con parametri espliciti e logging.
- Mantieni riferimenti a documentazione primaria quando usi comandi di sicurezza, boot o policy.

## Evidenze
- La fonte organizza comandi in ordine alfabetico con descrizione breve e link dedicati per approfondimenti ([SRC-20260417-a-z-index-windows-cmd-commands](../sources/SRC-20260417-a-z-index-windows-cmd-commands.md)).
